Project Overview

  • End-to-end graph learning inspired fraud detection workflow.
  • Backend + middleend + frontend integrated in one running app.
  • Research, diagnostics, and model governance included.

Why Graphs Matter

Fraud tends to hide in reuse patterns such as common devices, shared IPs, repeated merchants, and synchronized bursts.

What The Notebook Adds

Every major stage now has a visual explanation layer so results are readable without parsing raw JSON alone.

How To Read It

Move from profile, to graph shape, to training history, to model diagnostics, and then to prediction behavior.
